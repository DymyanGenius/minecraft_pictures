# Bigger Vanilla Story

Datapack для Java Edition мира `Новый мир`: добавляет сюжетную прогрессию, атмосферные события и небольшие survival-friendly награды без модов и ресурс-пака.

## Совместимость

- Пакет создан по структуре datapack из руководства Minecraft Wiki: корневой `pack.mcmeta`, папка `data`, пространства имен и теги `load`/`tick`.
- Используется современная структура Minecraft 1.21+: `data/<namespace>/function`, `data/<namespace>/advancement` и `data/minecraft/tags/function`.
- `pack.mcmeta` использует формат Minecraft Java 26.1.2: `min_format: [101, 1]` и `max_format: 101`, как в ванильных datapack-файлах этой версии.

## Что добавлено

1. **Сюжет из 7 этапов**: пробуждение мира, первый отголосок, Незер, кузница памяти, Око дороги, последнее небо и эпилог.
2. **Автоматическая прогрессия** по ванильным достижениям игрока:
   - получить жемчуг Эндера;
   - войти в Незер;
   - добыть стержень и древние обломки;
   - создать Око Эндера;
   - войти в Край;
   - победить дракона.
3. **Атмосфера**: звуки, частицы, титры и редкие сообщения по измерениям.
4. **Команды игрока**:
   - `/function bigger_vanilla:guide` — текущая цель;
   - `/function bigger_vanilla:claim` — небольшая награда главы.

## Проверка в игре

1. Открой мир `Новый мир`.
2. Выполни `/reload`.
3. Проверь `/datapack list`: должен появиться `[file/bigger_vanilla_story]`.
4. Выполни `/function bigger_vanilla:guide`.
