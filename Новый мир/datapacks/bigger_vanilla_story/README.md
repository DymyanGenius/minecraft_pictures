# Bigger Vanilla Story

Datapack для Java Edition мира `Новый мир`: превращает ванильное прохождение в полноценный сюжетный survival-режим с внутриигровым меню, торговцами-чертежниками, покупаемыми рецептами, диалогами через ответы игрока, псевдо-живыми разговорами жителей, катсценами, мини-боссом, секретами и двумя финалами без модов и ресурс-пака.

## Важные ограничения datapack

- Datapack не может заменить главное меню Minecraft до входа в мир и не может подключить настоящий внешний ИИ к чату без серверного мода/плагина.
- Вместо этого добавлено **внутриигровое подменю** после входа (`/function bigger_vanilla:menu/open`) и **диалоги через кликабельные `/trigger`-ответы**.
- Звуковое повествование реализовано ванильными `playsound`, титрами, эффектами и частицами.

## Совместимость

- Пакет создан по структуре datapack из руководства Minecraft Wiki: корневой `pack.mcmeta`, папка `data`, пространства имен и теги `load`/`tick`.
- Используется современная структура Minecraft 1.21+: `data/<namespace>/function`, `data/<namespace>/advancement`, `data/<namespace>/recipe` и `data/minecraft/tags/function`.
- `pack.mcmeta` использует формат Minecraft Java 26.1.2: `min_format: [101, 1]` и `max_format: 101`, как в ванильных datapack-файлах этой версии.

## Что добавлено

1. **Внутриигровое меню**: старт истории, призыв торговца и быстрый гайд через кликабельный интерфейс в чате.
2. **Торговцы-чертежники**: Архивариус Чертежей появляется командой меню и продаёт/разблокирует ключевые знания за изумруды; покупать можно только рядом с ним.
3. **Рецепты как прогрессия**: без покупки нужного чертежа ритуал или сюжетный этап не сработает, даже если ресурсы уже есть.
4. **Новые datapack-рецепты**:
   - `bigger_vanilla:whispering_index` — книга, аметист и чернила;
   - `bigger_vanilla:ember_rite` — огненный ритуальный крафт;
   - `bigger_vanilla:memory_lodestone` — поздний крафт памяти;
   - `bigger_vanilla:sky_lens` — линза дороги к Краю.
5. **Диалоги**: игрок отвечает Архивариусу кликами (`/trigger bv_reply set ...`), а жители периодически переговариваются рядом с игроком.
6. **Катсцены и звуковое повествование**: пролог, титры, darkness/slowness, частицы и звуки создают ощущение сцен.
7. **Режиссерский сюжетный маршрут**: пробуждение, жемчуг Эндера, ритуал шепчущего камня, Незер, Пепельный Страж, кузница памяти, Око дороги, Край и финальный выбор.
8. **Секретная линия древних городов**: осколок эха и пластинка 5 дают отдельные атмосферные события.
9. **Два финала**:
   - `/function bigger_vanilla:ending/restore` — восстановить мир;
   - `/function bigger_vanilla:ending/ascend` — вознестись вместе с пустотой.

## Основные команды

- `/function bigger_vanilla:menu/open` — открыть подменю datapack.
- `/function bigger_vanilla:market/spawn_archivist` — призвать Архивариуса Чертежей.
- `/function bigger_vanilla:guide` — текущая цель.
- `/function bigger_vanilla:claim` — небольшая награда главы.
- `/function bigger_vanilla:market/buy_whispering_stone` — купить рядом с Архивариусом первый чертёж за 3 изумруда.
- `/function bigger_vanilla:market/buy_ember_rite` — купить рядом с Архивариусом Пепельный ритуал за 5 изумрудов.
- `/function bigger_vanilla:market/buy_memory_forge` — купить рядом с Архивариусом Кузницу памяти за 7 изумрудов.
- `/function bigger_vanilla:market/buy_sky_lens` — купить рядом с Архивариусом Небесную линзу за 9 изумрудов.

## Проверка в игре

1. Открой мир `Новый мир`.
2. Выполни `/reload`.
3. Проверь `/datapack list`: должен появиться `[file/bigger_vanilla_story]`.
4. Выполни `/function bigger_vanilla:menu/open`.
